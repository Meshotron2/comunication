Projeto em Engenharia de Compudares e Informática

Meshotron é um projeto que visa a criação de um cluster de baixo custo para processamento de som em tempo real e realizar cálculos enormes. Paralelização acústica é um dos focos do projeto.

Descubra mais

Detalhes do Projeto (Objetivos)

Meshotron visa construir uma plataforma de baixo custo para demonstrar o conceito e facilitar o seu aperfeiçoamento.

  • Implementa DWM (digital-waveguide modelling), dividindo em pequenos blocos (data partitioning)

  • Usa topologia de malha 3D (cada unidade comunica com 6 vizinhos)

  • Idealmente atingir velocidades perto de análise em tempo real para salas não muito grandes.

  • Tornar a análise sonora de uma sala mais financeiramente acessível.

  • Efetua cálculos usando um algoritmo de forma optmizada e rápida

Tarefas Principais do projeto

Fase 1

Estudo da arquitectura e ambiente de desenvolvimento de unidades computacionais de baixo custo (e.g. RPi) e suas aplicações em computação paralela

Fase 2

Familiarização com o algoritmo de modelação acústica DWM 3D (ou outro algoritmo FDTD paralelizável de modo semelhante – crowd computing)

Fase 3

Selecção e programação de uma unidade computacional para executar o algoritmo num bloco do modelo acústico e estabelecer links de comunicação de dados com 6 unidades idênticas alocadas a blocos adjacentes

Fase 4

Interligação de um pequeno conjunto de unidades a computador host (e.g. por Ethernet – vide figura) para download do modelo particionado (um bloco a cada unidade) e injecção/recolha dos sinais áudio de excitação/resposta. Finalmente realizar o teste do sistema
